# Acerca del Proyecto
App-web para un sistema POS para Restaurantes, restaurante con menú del día, con sistema de Mesas, Pedidos y Pedidos en Lista para la cocina, sistema moderno para el apoyo de esta gran herramienta ofrecer un buen servicio a los clientes


### Instalación

```
#instalar dependencias

npm install
```

```
#iniciar el proyecto local

npm run dev
```

---

### Tecnologías

- Next.js + React
- Typescript
- Subapase
- TaiwlindCSS
